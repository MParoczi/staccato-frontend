Luminance-based contrast color helper (inline, pure function):
```typescript
function getContrastColor(hex: string): string {
  const r = parseInt(hex.slice(1, 3), 16)
  const g = parseInt(hex.slice(3, 5), 16)
  const b = parseInt(hex.slice(5, 7), 16)
  const luminance = (0.299 * r + 0.587 * g + 0.114 * b) / 255
  return luminance > 0.5 ? '#1a1a1a' : '#ffffff'
}
```

Component: `export default function NotebookCoverPage()`:
- Get notebook from outlet: `const { notebook } = useOutletContext<NotebookOutletContext>()`
- Compute text color: `const textColor = getContrastColor(notebook.coverColor)`
- Return a full-bleed div that fills available height: `className="flex min-h-[70vh] flex-col items-center justify-center px-8 py-16"` with `style={{ backgroundColor: notebook.coverColor, color: textColor }}`.
