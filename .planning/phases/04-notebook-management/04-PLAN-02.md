```typescript
function extractErrorMessage(error: unknown, fallback: string): string {
  if (error && typeof error === 'object' && 'response' in error) {
    const resp = (error as { response?: { data?: { detail?: string } } }).response
    if (resp?.data?.detail) return resp.data.detail
  }
  return fallback
}
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User input → POST/PATCH payload | Form values (title, coverColor, pageSize, stylePreset) sent as JSON to backend |
| API response → rendered JSX | Notebook title and other fields rendered as React text nodes |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-04 | XSS | Notebook title in NotebookCard JSX | mitigated | React escapes all JSX text interpolation; `{notebook.title}` is safe; no dangerouslySetInnerHTML |
| T-04-05 | XSS | Notebook title in NotebookFormDialog Input | mitigated | react-hook-form controlled input; value is a string passed to API, never eval'd or injected as HTML |
